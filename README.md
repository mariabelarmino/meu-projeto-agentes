# Projeto de Agentes com FastAPI

Este repositório contém o estudo de Orientação a Objetos aplicada a agentes e uma API para execução de tarefas.

## Como rodar
1. Crie o venv: `python3 -m venv venv`
2. Ative: `source venv/bin/activate`
3. Instale: `pip install -r requirements.txt`
4. Rode: `uvicorn main:app --reload`
